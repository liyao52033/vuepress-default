# 谷歌重大更新：国内手动开启 Gemini AI 侧边栏与 Auto Browse 自动浏览全攻略

## 🛠️ 功能亮点
Google Chrome 集成 Gemini 3 模型后，已从单一工具转变为智能助手：
* **Gemini 侧边栏 (Side Panel)：** 无需切换标签页即可总结内容、对比评论或同步日历。
* **Auto Browse (自动浏览)：** 具备代理能力 (Agentic Action)，可代劳预订、填表、收集文件或智能购物。
* **Nano Banana：** 支持通过提示词直接在线修改网页图像。
* **个人智能化：** 具备长期记忆，根据历史对话提供个性化建议。

---

## 第一步：修改 Local State 配置文件

**注意：操作前请彻底关闭 Chrome 浏览器并备份该文件。**

### 1. 文件路径
* **Windows:** `%LOCALAPPDATA%\Google\Chrome\User Data\Local State`
* **macOS:** `~/Library/Application Support/Google/Chrome/Local State`
* **Linux:** `~/.config/google-chrome/Local State`

### 2. 编辑内容
使用文本编辑器（如记事本、VS Code）打开 `Local State` 文件，搜索并修改以下三项：
* `variations_permanent_consistency_country`: 将值由 `["版本号","cn"]` 改为 `["版本号","us"]`。
* `variations_country`: 将值由 `"cn"` 改为 `"us"`。
* `is_glic_eligible`: 将值由 `false` 改为 `true`。

---

## 第二步：配置 Chrome Flags 标志位

1. 在 Chrome 地址栏输入：`chrome://flags`
2. 搜索以下项并将其状态从 `Default` 修改为 **`Enabled`**：
   * `#glic`
   * `#glic-side-panel`

---

## 第三步：语言与环境设置

1. **修改首选语言：** * 进入 Chrome 设置 -> **语言 (Languages)**。
   * 添加 **English**，点击右侧三个点，选择“置顶”并设置为浏览器显示语言。
   * *注：若不使用英文界面，可能仅能使用“帮我写 (Help me write)”功能，无法开启完整智能体。*
2. **网络环境：** 必须确保处于**美国节点**网络环境下运行。
3. **重启浏览器：** 重启并登录 Chrome 后，右上角会出现 **Gemini 图标**，点击即可开启侧边栏。

---

## 💡 自动化进阶工具
如果你熟悉编程，可以使用 GitHub 上的开源 Python 脚本一键完成配置：
* **项目名称：** [enable-chrome-ai](https://github.com/lcandy2/enable-chrome-ai)
* **功能：** 无需清除数据，自动修改本地 `Local State` 配置以启用浏览器内置 AI。

---

## 总结
1. 修改 `Local State` 文件中的国家代码和 Glic 权限。
2. 开启 `chrome://flags` 中的 Glic 相关配置。
3. 保持 Chrome 语言为 **English**。
4. Gemini AI 侧边栏必须使用**美国线路**节点。
5. Gemini网页版还无法访问的话新建一个[Gem](https://gemini.google.com/gems/create),左侧输入名称，右侧发起会话，然后访问[https://gemini.google.com/app](https://gemini.google.com/app)即可
