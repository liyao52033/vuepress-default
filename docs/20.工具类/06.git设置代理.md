1、打开clash开启代理

2、系统设置代理

​    网络和Internet设置 -> 代理 -> 手动设置代理

3、git设置代理

```bash
git config --global http.proxy http://127.0.0.1:7890
```

4、查看git代理是否设置成功

```bash
git config --global -l
```

5、git取消代理

```sh
 git config --global --unset http.proxy
 git config --global --unset https.proxy
```





> .gitignore 新加忽略文件，发现忽略的文件还会提交。

```
git rm -r --cached .
git add .
git commit -m 'update .gitignore'
```
