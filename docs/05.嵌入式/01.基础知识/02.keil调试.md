在工程配置菜单中，User选项卡，编译后执行的命令，设置为 

```bash
fromelf --bin -o "$L@L.bin" "#L"
```

当然也可以使用上面那种绝对路径的方式，需要看指定fromelf文件的路径，输出BIN文件的路径和生成的AXF文件的路径。
